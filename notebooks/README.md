# Notebooks Directory

This folder contains Jupyter notebooks for exploration and experimentation.

> **Note:** Notebooks are for development/research only. Production code should be in `src/`.

## Usage

```bash
# Install Jupyter
pip install jupyter

# Start Jupyter
jupyter notebook
```

## Guidelines

1. **Keep notebooks clean** - Clear outputs before committing
2. **Extract reusable code** - Move production-ready code to `src/`
3. **Document findings** - Add markdown cells explaining your analysis
4. **Use relative imports** - Reference `src/` modules properly
