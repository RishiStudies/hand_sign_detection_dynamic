# Data Directory

This directory contains training and evaluation datasets for the Hand Sign Detection system.

> **Note:** Data files are NOT committed to Git. Download or generate them locally.

## Required Datasets

### 1. WLASL Dataset (American Sign Language)

Download the WLASL dataset for dynamic sign recognition:

```bash
# Option 1: Official source
# Visit: https://dxli94.github.io/WLASL/

# Option 2: Using provided scripts
python scripts/download_wlasl.py --classes 100
```

Expected files after download:
- `WLASL_v0.3.json` - Video annotations
- `nslt_100.json`, `nslt_300.json`, etc. - Class subsets
- `videos/` - Downloaded video files

### 2. Hand Alphabet Dataset (Static Letters)

For static fingerspelling recognition:

```bash
# Generate from training pipeline
python -m hand_sign_detection.training.pipeline train-alphabet
```

Expected files:
- `hand_alphabet_data.csv` - Feature data for A-Z letters

### 3. Processed Feature Files

Generated by the training pipeline:

```bash
# Extract features from videos
python -m hand_sign_detection.training.pipeline extract-features
```

Expected files:
- `X_data.npy` - Feature vectors
- `y_data.npy` - Labels

## Directory Structure

```
data/
├── README.md          # This file
├── videos/            # WLASL video files (gitignored)
├── WLASL_v0.3.json    # Video annotations (gitignored)

├── X_data.npy         # Extracted features (gitignored)
├── y_data.npy         # Labels (gitignored)

```

## Troubleshooting

**Missing data files?**
1. Run the training pipeline: `python -m hand_sign_detection.training.pipeline --help`
2. Check internet connection for downloads
3. Ensure sufficient disk space

**Video download failures?**
- Some WLASL videos may be unavailable (YouTube removals)
- Check `missing.txt` for failed downloads
- Use `--retry-failed` flag to retry
